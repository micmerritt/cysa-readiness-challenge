# CySA+ Readiness Challenge

## Purpose

The CySA+ Readiness Challenge is a scenario-based diagnostic designed to evaluate reasoning ability before attempting the CompTIA CySA+ exam.

This is **not** a practice exam clone. It is **not** CertMaster. It does not reuse proprietary material.

It is designed to answer one question:
> Can you reason through new cybersecurity scenarios under pressure?

---

## What This Tool Tests

The challenge focuses on four core reasoning areas:

- **Log Analysis**
- **Incident Response Sequencing**
- **Vulnerability Prioritization**
- **Detection Reasoning**

Questions are scenario-heavy and decision-based. They emphasize interpretation, prioritization, and next-step logic rather than trivia or tool memorization.

---

## How It Works

1. Start the challenge.
2. The system generates a unique attempt using a seed.
3. Complete 25 scenario-based questions.
4. Review your results.
5. Click **Copy Results Summary** and send it to your instructor.

Each attempt:
- Uses a blueprint across categories (7/6/6/6)
- Shuffles answer order
- May vary scenario details using template variables
- Includes a reproducible seed

---

## Results & Tiers

You will receive:

- Overall score
- Category breakdown
- Readiness tier:
  - **Ready**
  - **Borderline**
  - **Not Ready**
- Targeted remediation guidance

This tool is a readiness diagnostic. It does **not** guarantee exam success.

---

## Instructor Workflow

Students should send:

- Results summary
- Seed
- Tier outcome

The seed allows the attempt to be reproduced if needed. This diagnostic supports informed readiness conversations before voucher approval.

---

## Design Philosophy

- Static, browser-based

The results summary includes:
- Date/time
- Application version

If the question content changes materially, the application version will be incremented.
